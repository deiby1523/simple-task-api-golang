# Go Task API

A simple RESTful API built with **Go (Golang)** for managing tasks.  
This project is intended for learning purposes and focuses on understanding how to build APIs using Go.

## Features

- Create tasks  
- List all tasks  
- Get a task by ID  
- Update tasks  
- Delete tasks  

## Tech Stack

- Go (standard library)  
- `net/http`  
- `encoding/json`
